# TrabPratPA

## Tabuleiro
  6 Linhas por 7 Colunas.
  Ganha quem puser 4 peças na vertical, horizontal e diagonal.
  
## Inicio do Jogo
  - Nome do Jogador(têm de ser nomes diferentes)
  - Sorteio de quem joga primeiro
  
  ### Após 8 jogadas (ou 4 para o jogador):
    - Minijogo
      - Pode aceitar ou não o minijogo.
      - As atividades sao alternadas entre:
        - Calculos de apenas 2 casas decimais com (+,-,*).
        - Jogo das Palavras com 100 palavras num ficheiro.
    - Se acertar ganha uma peça que faz desaparecer as peças dessa coluna.
    - Se o jogador ganhar ainda tem possibilidade de jogar. Se perder a jogada passa.
    - Contagem das jogadas para os minijogos
## Armazenar jogadas
  - Armazenar as jogadas para puder voltar atrás
  - Créditos:
    - jogada anterior -1 crédito.
    - Sendo dados 5 créditos iniciais.
    - A jogada desfeita vai sempre para a jogada anterior.
  - Cada vez que volta atrás e havia um minijogo volta para a joga anterior ao mini jogo

## Replay
  - Histórico de 5 jogos
  - Guardar o resultado dos minijogos.
